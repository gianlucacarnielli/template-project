Update Documentation

Edit Existing Page

To update an existing page follow the steps below:
  1. Edit the live script corresponding to the documentation page of interest.
  2. Build the documentation using the relative project shortcut.

Add New Page

To add a new page to the documentation follow the steps below:
  1. Create a new live script in the project folder named "LiveScriptAndHTML";
  2. Add the newly created live script to the project;
  3. [Optional: change the project label of the added file to "Documentation"];
  4. Add a toc item in "helptoc.xml" (the HTML file name will be the same as that of the exported live script);
  5. Build the documentation using the relative project shortcut.
