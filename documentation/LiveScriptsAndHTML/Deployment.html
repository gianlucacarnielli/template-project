Deployment

Standalone App

To deploy the application as a standalone executable, press the "Deploy Standalone App" project shortcut. This will automatically create a standalone executable in the folder deployment/standalone. To run the standalone executable download the MATLAB Runtime.

Web App

To deploy the application as a web app, proceed as follows:
  1. Install MATLAB Web App Server (or the development version in MATLAB Compiler)
  2. In deployment.json (in the deployment folder) specify the server folder to which deploy the web app, namely the CTF archive.
  3. Run the project shortcut "Deployment Web App". The Compiler should automatically identify all dependencies and include them in CTF archive.

About deployment.json

This file allows to store deployment-related information, such as output directories, additional required files, and compilation options.
